Usage

Sending a template requires a Ping instance.

import net.gutefrage.mandrill._
import net.gutefrage.mandrill.core._

val apiKey = MandrillApiKey("your-api-key")
val ping = Mandrill(apiKey).users.ping

The ping has the required shape for the Mandrill API. Serialize with your own serializer or with one of the supplied by Scala Mandrill and send the json to the required REST endpoint. Mandrill returns either a generic error or PingResponse. gain you can provide your own serializer or use the ones provided by Scala Mandrill.
